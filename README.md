# Markedit Backend.

Backend for Markedit: the simple markdown note taking app.

## Usage

Rename file **config.env** to **.env** and update the values.

### .env example

```
# Port to use
PORT=5000


# Maximum number of requests per minute (only used in prod mode)
RATE_LIMIT=100
```
